# 项目简介

## 核心功能
- 基于YOLOv8的停车位检测系统，支持识别占用和空闲两类停车位
- 支持多种输入源：单张图片、图片文件夹、视频文件及本地摄像头
- 提供实时显示功能：目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 可自定义检测参数：置信度阈值、IOU阈值
- 支持选择不同检测模型
- 提供结果保存功能

## 适用场景
- 停车场管理系统开发
- 智能停车解决方案研究
- 计算机视觉学习项目
- 停车位状态监控系统

## 技术特点
- 采用YOLOv8目标检测算法
- 使用PyQt5构建用户界面
- 支持Python3.9及OpenCV、PyTorch等深度学习框架
- 包含1000张标注数据集，涵盖占用和空闲两种状态
- 提供完整的源码、训练好的模型及运行教程
- 兼容PyCharm和VSCode开发环境
